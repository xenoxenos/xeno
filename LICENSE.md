## 🛠️ Troubleshooting Common Issues

* **Error: App Not Installed**
  * *Description*: The Android package manager rejects the target configuration file.
  * *Solution*: **Ensure that any old developer builds are completely deleted and that you have enabled third-party installations.**

* **Error: Attachment Failure**
  * *Description*: The internal script engine does not hook into the target application thread.
  * *Solution*: **Restart the app, clear the active device memory, and launch the application before attaching.**

* **Error: Workspace Permission Blocked**
  * *Description*: The console throws errors when attempting to read or write files.
  * *Solution*: **Check your application manager and verify that file storage access permissions have been granted.**

* **Error: Screen Freezes on Start**
  * *Description*: The UI becomes completely unresponsive when launched.
  * *Solution*: **Adjust your `config.json` parameters to set a lower screen refresh rate and clear your device's cache.**
